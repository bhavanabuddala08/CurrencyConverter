# Read Me First
The following was discovered as part of building this project:

* Spring Cloud Gateway requires Spring WebFlux, your choice of Spring Web has been replaced accordingly.
* The original package name 'com.hcltest.springboot.microservice.currency-exchange' is invalid and this project uses 'com.hcltest.springboot.microservice.currencyexchange' instead.
